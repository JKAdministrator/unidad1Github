• q

    Para salir cuando un comando esta mostrando algo y necesitas volver a la consola

  • git init

    Inicializa un repositorio local

  • git status

    Status del repositorio. Indica cuales estan sin tracker (en rojo, no los tiene en cuenta el versionador) y en que branch estamos

  • git add .

    . es todo el directorio. sino puedo poner el nombre del archivo. Agrega el archivo al versionador

  • git commit -m "esto es un commit"

    . es todo el directorio. sino puedo poner el nombre del archivo. Agrega el archivo al versionador

  • git branch

    muestra todos los branchs y en cual estamos

  • git branch nameOfBranch

    crea un nuevo branch con el nombre nameOfBranch. No mueve al branch creado

  • git checkout nameOfBranch

    te mueve al branch nameOfBranch

  • git merge nameOfBranch

    hace un pull desde el branch nameOfBranch al actual. Ej: mover todo lo que hay en el branch hotfix1 al master, integrando los cambios hechos en hotfix_1 al fuente del master

  • git log

    log de commits
